## Overview

Leonid 91 empowers you to create blazing-fast, SEO-optimized websites with minimal effort. Developed by Agenc.io, it combines modern build tools with intelligent optimizations to ensure your site ranks high on search engines and loads in under 2 seconds. You focus on content while Leonid 91 handles performance and discoverability.

Whether you run a small business or an agency serving multiple clients, Leonid 91 streamlines your workflow. It generates static sites that scale effortlessly, reducing hosting costs and improving user experience.

<Callout kind="info">
  Leonid 91 uses advanced bundling and tree-shaking to achieve sub-100KB payloads for most sites.
</Callout>

## Quick Start

Get your first site live in under 5 minutes.

<Steps>
  <Step title="Install Leonid 91" icon="download">
    Use npm or yarn to add the CLI globally.

    <CodeGroup tabs="npm,yarn">
    ````bash
    npm install -g @agenc.io/leonid-91
    ````
    ````bash
    yarn global add @agenc.io/leonid-91
    ````
    </CodeGroup>
  </Step>
  <Step title="Create Project" icon="plus">
    Initialize a new site with your choice of template.

    ````bash
    leonid init my-site --template business
    cd my-site
    ````
  </Step>
  <Step title="Build and Preview" icon="play">
    Build optimized assets and start a local server.

    ````bash
    leonid build
    leonid preview
    ````
  </Step>
</Steps>
