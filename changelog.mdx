<Update label="2024-10-15" description="v1.1.0" tags={["feature", "improvement"]}>

## New Features

- Added automatic SEO meta tag generator for pages, optimizing titles and descriptions based on your content.
- Introduced sitemap.xml auto-generation with dynamic URL crawling for better search engine indexing.

## Improvements

- Enhanced Core Web Vitals scores by optimizing image lazy-loading and font preloading.
- Reduced initial page load time by `>20%` through improved CSS critical path rendering.

## Bug Fixes

- Fixed intermittent 404 errors on dynamic routes during high-traffic deploys.
- Resolved mobile viewport scaling issues on iOS Safari.

</Update>

## Upgrade Process

Follow these steps to update your Leonid 91 sites safely.

<Steps>
  <Step title="Backup Site" icon="database">
    Export your current configuration:

````yaml
# config.yaml
site:
  name: "Your Business Site"
  domain: "example.com"
  seo:
    title: "Your Business"
````

  </Step>
  <Step title="Update CLI" icon="download">
    <CodeGroup tabs="npm,yarn">
````bash
npm install -g @agenc.io/leonid-cli@latest
````
````bash
yarn global add @agenc.io/leonid-cli@latest
````
    </CodeGroup>
  </Step>
  <Step title="Deploy" icon="rocket">
    Run `leonid deploy` in your project root.
  </Step>
</Steps>

<Callout kind="tip">
  Test updates on a staging site first at https://staging.example.com.
</Callout>
